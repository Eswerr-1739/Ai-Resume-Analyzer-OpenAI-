**AI Resume Analyzer** is an end-to-end, AI-powered tool that helps users analyze resumes against job descriptions.  
It provides **similarity scores, skill matching, section-level scoring**, and generates **CSV/JSON reports**, with an interactive **web dashboard** built using Streamlit.  

This project is **open-source**, free, and suitable for showcasing AI and NLP skills on GitHub.

---

## Features

- **Flexible Resume Input**: Supports `.pdf`, `.docx`, `.txt` files.  
- **Job Description Matching**: Compute resume-JD similarity.  
- **Skill Matching**: Detect skills automatically using fuzzy matching and optional LLM evaluation.  
- **Section Scoring**: Analyze sections like Education, Experience, Skills individually.  
- **Reports**: Export results as CSV and JSON files.  
- **Interactive Dashboard**: Real-time analysis via Streamlit UI.  
- **LLM Integration**: Optional AI-powered skill extraction and recommendation using OpenAI API.
